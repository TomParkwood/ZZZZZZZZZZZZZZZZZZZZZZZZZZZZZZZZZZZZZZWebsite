The A4 Colour range of Utax machines offer a small but sturdy desktop MFP, which packs all of the features and functionality of machines twice its size.

With printing speeds of up to 26 pages per minute and scanning speeds of up to 30 pages per minute, it will keep up with the much bigger machines.

As well as the standard features you may expect from an MFP this model also packs faxing, Wi-Fi and a free mobile app which makes printing from iOS and Android simple.

3. Card Authentication Kit (B) Allows for user authentication with ID cards at the machine.
4. UG-33 ThinPrint Option, allows for printing without a print driver.
5. SD / SDHC Memory Card Used for storing fonts, macros and overlays.
6. USB Keyboard Used to enter information on the operation panel, a mount is also available to install the keyboard onto the machine.
