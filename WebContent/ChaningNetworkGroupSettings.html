If your SMB scans cannot get through to your shared folder, or nobody on your network can see your machine. Your network settings maybe wrong.

Here's how to change your network group settings.

{slider title="Windows 7" open="false" alias="Windows7"}

One way to change this is to open control panel1, then go into network and internet, then finally into network and sharing centre2.
From here it will show the network you are connected to. It will show the name of the network then either Public, Work or Home network3. If the network is set to public it will not allow for any other devices on the network to connect to it, thus blocking the scans.
So change this to an appropriate one, then to check that it is working. Open up My Computer, then on the left select network, this will show all of the devices on the network as well as the printer4..
So try sending a scan to the folder again.

{slider-embed title="Images" open="false" alias="W7Images"}

{/sliders-embed}

{slider title="Windows 10" alias="Windows10"}
One way to change this is to open the settings menu from the start menu. Then click on Network & Internet.
If you connect to the internet wirelessly select Wi-Fi or if you use a ethernet cable select ethernet.
Click on the connection name which is followed by connected. At the top there should be a switch which says Find Devices and content, turn this on.
Now try sending a scan to the folder again, it should go through if it was the network settings which were wrong.
