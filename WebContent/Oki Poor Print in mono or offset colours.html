We have found a common problem with a lot of OKI printers, that use the LED print system.
Sometimes the mono print will almost disappear or become somewhat hieroglyphic, we also see the colours either becoming offset to each other or completely vanishing.
This fix is simple.

  1. Turn the machine off and remove the power lead!
  2. Open the cover which shows the drums and toners.
  3. Look under the open cover and you will see four black led strips.
  4. Note the ribbon cable on the right side going into these bars.
  5. Carefully pull the ribbon cable from the led bar.
  6. Reinsert the cable back into the bar, checking that it is seated securely.
  7. Restart the machine and test.
