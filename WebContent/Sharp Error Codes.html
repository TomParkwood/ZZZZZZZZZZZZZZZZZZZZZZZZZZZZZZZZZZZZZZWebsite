Here is a list of error codes which could show on your machine and how to resolve them.

Maintenance Counter

Code Description How To Resolve The Problem
TA Maintenance Counter Total Reached This can only be solved by replacing the parts which have reached their life then resetting the count, or by resetting the count using simulation codes.
CA Maintenance Counter Colour Reached This can only be solved by replacing the parts which have reached their life then resetting the count, or by resetting the count using simulation codes.
AA Maintenance Counter Of Both Total And Colour Reached This can only be solved by replacing the parts which have reached their life then resetting the count, or by resetting the count using simulation codes.

Code Description How To Resolve The Problem

FK2 Pressure Roller Print Counter Has Reached Life The only way to make this message disappear is to either replace the pressure roller in the fuser (although it is recommended to replace everything in the fuser at once not just the pressure roller) and reset the count, or reset the count by using simulation codes. You can ignore this message if you want to, but we recommend replacing the parts when the message appears so you can keep the quality of the prints to a high standard.
FK3 Fusing Web Print Counter Has Reached Life The only way to make this message disappear is to either replace the fusing web (not all machines have a fusing web) (although it is recommended to replace everything in the fuser at once not just the fusing web) and reset the count, or reset the count by using simulation codes. You can ignore this message if you want to, but we recommend replacing the parts when the message appears so you can keep the quality of the prints to a high standard.
FK3 (Pop-up) If FK3 has popped up and not just in the top left then the fusing web has reached the end. The only way to make this message disappear is to either replace the fusing web (not all machines have a fusing web) (although it is recommended to replace everything in the fuser at once not just the fusing web) and reset the count.
